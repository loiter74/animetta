01Title bar

44 px tall glass strip. Drag-region runs across the empty middle. The right-side status pill is a mono string — never wraps, never grows past 18 ch.

02Persona drawer

Pinned 280 px column with the active persona on top, then the settings list (each row tagged with its section icon), then a memory feed. Background sits at 45% alpha so the scene shows through.

03Live2D stage

Center-fluid column. The character is anchored to the lower-center third. Stage controls float top-right; the scene tag floats top-left. Bilingual subtitle docks above the bottom edge.

04Chat panel

340 px right column. AI bubbles ride the left, user bubbles ride the right; the speaker is identified by tail corner more than by alignment. Day stamps every ~30 min.

05Input bar

Always shows the focus ring while the panel is the active surface, even when the field is empty — Anima is always ready to listen. The mic button sits inside the field; the send button sits outside.

06Status badge

Tiny green dot + LIVE. Switches to amber with the latency number if > 200 ms, red with OFFLINE if the websocket drops.
