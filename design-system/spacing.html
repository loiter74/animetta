Spacing, radii, motion

Animetta inherits its rhythm from Tailwind / UnoCSS — a 4-pixel base. Radii are deliberately generous (the smallest button is already 12px), and motion is fast and springy. Stiff or slow easing reads as "broken," not "elegant," in a companion product.

Spacing scale

A 4-pixel base with a 6-pixel half-step. Most component padding lives at 12px or 16px; component gaps live at 8px or 12px. Anything above 32 px is layout breathing room.

Radii

Animetta is round. There are no 90-degree corners in the system other than the window itself. The default for any new component is --r-lg (12 px).

Motion

Three durations. Two easings. Anything else is a custom animation that should be reviewed.

Durations

fast
--d-fast
150 ms
base
--d-base
200 ms
slow
--d-slow
300 ms

fast for button presses and hovers (no perceptible delay). base for the vast majority of transitions: tab switches, panel open/close, focus rings. slow only when something is physically moving across a meaningful distance, e.g. the settings drawer sliding in from the right.

Easings

--ease-out-expo · default --ease-back-soft · entry

out-expo handles 90% of cases — a fast initial move that settles softly. back-soft adds a tiny overshoot, used when a new element enters from off-screen (notification toast, new bubble) so it feels alive instead of dropped.
